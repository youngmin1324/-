# 지렁이 키우기 (snake 응용 게임)

# 게임 설명
요리조리 움직이는 굶주린 새(◎)를 피해 지렁이(○○○○○)를 키워보자!  
영양가 있는 먹이(∞)를 먹으면 지렁이의 몸을 키울 수 있다.  
그러나, 독이 든 먹이(‡)를 먹으면 몸이 줄어 든다.

약골 지렁이라 벽에 부딪히면 죽는다.  
새에게 잡아먹히거나 독이 든 먹이를 많이 먹어 몸이 사라지면 당연히 죽는다.

# 게임 방법
방향키를 이용해 지렁이가 움직이는 방향을 바꿀 수 있다.
